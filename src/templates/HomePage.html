BOOK YOUR LIFE


This project is a web application that will recommend books to the user based on a selection of books already read.

How it works

We use a Latent Dirichlet Allocation (LDA) model to find the topics of the books selected. Then we use the Euclidean distance to find the books that are closer to the ones selected, and therefore the most similar.

How to use

It is pretty straight forward:
  1. Click on get a recommendation on the menu
  2. Select the books you have read
  3. Click on get recommendation
